# Synthetic Django Repo Snapshot

This fixture is a compact static source snapshot for future
Django-in-Electron wrapping work. The current runner validates it as pack-local
fixture metadata only. It is not copied, executed, mutated, installed, scored,
or injected into case prompts.

The files describe a tiny server-rendered inventory dashboard with a health
endpoint, static CSS, and portable relative settings. The snapshot intentionally
omits generated files, dependency lockfiles, databases, media uploads, caches,
virtual environments, and host-specific paths.
